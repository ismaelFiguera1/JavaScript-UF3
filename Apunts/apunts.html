Seleccionar elements DOM

Hi ha la manera antiga i la nova:
La antiga:

  • Per tag: document.getElementsByTagName("li")
  • Per classe: document.getElementsByClassName("card")
  • Per atribut name: document.getElementsByName("nom")
  • Per id: document.getElementById("menu")

La nova:

  • document.querySelector(".cards"): Aquest agafa el 1r element que tingui la classe cards
  • document.querySelectorAll("a"): Aquest agafa TOTS els elements que tinguin el tag "a"

Recorrer Nodes

Ara recorrerem els nodes, els nodes son els elements que hem seleccionat del DOM
Exemples:

  • console.log(document.querySelectorAll("h2").forEach((ele)=>{ console.log(ele);}));
  • console.log(document.querySelectorAll(".tema").forEach((ele)=>{ console.log(ele);}));
  • console.log(document.querySelectorAll("li").forEach((ele)=>{ console.log(ele);}));
